## General info
This project is a dashboard comparing crime with different social criteria in the world.
	
## Technologies
Project is created with:
* Python
* Streamlit
* Pandas
* Numpy
* Mapclassify
* Matplotlib
* Geopandas
* Plotly.express
* Plotly
	
## Setup
To run this project :

```
$ streamlit run dashboard.py
```
